# Fictional University Theme

Custom WordPress theme built from scratch as part of a WordPress Developer course.

## Features
- Custom Post Types (Events, Professors, Programs)
- Template hierarchy implementation
- Archive and single templates
- Custom queries with WP_Query
- REST API integration
- Live search (if you implement later)

## Technologies
- WordPress
- PHP
- JavaScript
- CSS
- REST API

## What I Learned
- WordPress template hierarchy
- Custom Post Types and Taxonomies
- Working with WP_Query
- Basic REST API usage
- Theme structure and organization
